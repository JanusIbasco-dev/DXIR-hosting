# DXIR Hosting

A clean, static demo website for a Minecraft server hosting brand.

## Features
- Professional dark UI with blue and purple accents
- Landing page with pricing cards
- Register / login flow without verification
- User dashboard for GCash payment instructions and proof upload
- Admin-only dashboard with account and submission review
- Responsive layout for desktop and mobile
